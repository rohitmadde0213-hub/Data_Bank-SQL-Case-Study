💾 **Data Bank – SQL Case Study**
📌 Project Overview
  The financial industry is rapidly evolving with the emergence of **Neo-Banks** — **fully digital banks** that operate without physical branches.
  Inspired by this innovation, Danny envisioned a unique intersection between:
💳 **Digital Banking, ₿ Cryptocurrency**, 📊 **Data Analytics**

  This led to the creation of Data Bank — a next-generation platform that combines traditional banking services with a secure, distributed data storage system.

🚀**What is Data Bank?**

**Data Bank functions like a digital bank, but with a twist:**

  👉 Customers not only manage their finances
  👉 They are also allocated cloud data storage

🔗 **Key Concept**

  A customer’s data storage limit is directly proportional to their account balance.
  
  This introduces interesting challenges in:

📈**Capacity planning**
  📊 Customer growth analysis
  🧠 Data-driven forecasting
  🎯 Business Problem

**The Data Bank management team aims to**:

  Increase the total customer base
  Estimate future data storage requirements
  Track usage trends and growth patterns
  
  To achieve this, they require data-driven insights to:
  
  Optimize infrastructure
  Forecast storage demand
  Support strategic decision-making

🧩 **Case Study Objective**

  This case study focuses on:
  
  📊 **Calculating key business metrics
  📈 Analyzing customer growth
  🔍 Understanding storage usage behavior
  🧠 Generating insights for future planning
  🗂️ Available Data**

**The Data Bank team has provided:**
  A structured data model
  Sample datasets to understand:
  Customer accounts
  Transactions
  Storage allocation
  
  These datasets will be used to perform analysis and derive actionable insights.

🛠️ **Skills & Tools Used**
  SQL (Advanced Queries, CTEs, Window Functions)
  Data Analysis
  Business Problem Solving
  Data Modeling Concepts

📈 **Expected Outcomes**

  By completing this case study, you will be able to:

  Analyze customer and transaction data effectively
  Estimate future storage requirements
  Provide actionable recommendations
  Build a strong data storytelling approach
